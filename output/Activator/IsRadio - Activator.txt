'''Member of:''' [[Activator Script]]

Returns whether this Activator is a radio or not.

== Syntax ==
<source lang="papyrus">
bool Function IsRadio() Native
</source>

== Parameters ==
*None

== Return Value ==
*True if this Activator is a radio.

== Examples ==
<source lang="papyrus">
If myObject.IsRadio()
    Debug.Trace("My object is a radio.")
EndIf
</source>
